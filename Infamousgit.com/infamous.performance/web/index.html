INFAMOUS

CM11 THEME

Infamous CM11 Theme is a clean minimal blacked out theme for custom Cyanogen Mod type ROMs.

Custom User interface

The app itself has many awesome features. You can apply the theme right from the app. It has a preview gallery as well as wallpapers that you can apply right from the app.

Requests

You can request apps to be themed from the app and it will be sent to me to work on.

Infamous extra's

Many other great features include a link to my development team and our forum and other sites.

Contact the developer

You get a direct line to the developer (me.) Email any suggestions you have as well as submit wallpapers to be added.
